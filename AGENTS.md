# Mantis - Free React Admin Dashboard

A free and open source React dashboard template built with Material UI v7, designed for flexibility and customizability. Combines Material Design and Ant Design principles.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 7
- **UI Library**: Material UI v7 (MUI)
- **Styling**: Emotion CSS-in-JS
- **Routing**: React Router v7
- **Data Fetching**: SWR
- **Forms**: Formik + Yup validation
- **Animations**: Framer Motion
- **Charts**: MUI X Charts
- **Package Manager**: Yarn 4

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy this React admin dashboard to Vercel or Netlify |

## Project Structure

```
src/
├── assets/           # Static assets (images, etc.)
├── components/       # Reusable UI components
├── contexts/         # React context providers
├── hooks/            # Custom React hooks
├── layout/           # Layout components (sidebar, header)
├── menu-items/       # Navigation menu configuration
├── pages/            # Route pages
├── routes/           # Route definitions
├── themes/           # MUI theme configuration
└── utils/            # Utility functions
```

## Key Commands

```bash
yarn install         # Install dependencies
yarn start           # Start dev server (port 5173)
yarn build           # Production build
yarn preview         # Preview production build
yarn lint            # Run ESLint
yarn lint:fix        # Fix ESLint issues
yarn prettier        # Format code
```

## Features

- **Dashboard**: Analytics overview with charts and statistics
- **Authentication**: Login/Register pages with form validation
- **Components**: Buttons, tables, forms, cards, and more
- **Responsive**: Mobile-first design
- **Theming**: Customizable color schemes

## Customization

- Theme configuration in `src/themes/`
- Menu items in `src/menu-items/`
- Layout components in `src/layout/`
- Add new pages in `src/pages/`
